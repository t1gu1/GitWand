# GitWand v1.6.2 — Image diff viewer + Folder tree view

Two features land in this release, both aimed at making large commits and binary assets readable at a glance.

## What's new

### 🖼️ Image diff viewer

Binary image files in a commit diff now render side-by-side instead of showing "Binary file changed". Four comparison modes:

- **Side-by-side** — before/after with dimension + size metadata
- **Overlay** — stacked images with an opacity slider
- **Blink** — alternates the two images to reveal pixel-level deltas
- **Slider** — draggable reveal between before/after

Supports **PNG, JPG, WebP, GIF, SVG** (SVG renders as raster, not raw XML, so you see the actual image — but the text diff is still available via the file history). GIF files get an "animated" badge. Images over 20 MB per side are skipped with an explicit notice to keep the viewer snappy.

Under the hood: a new `read_file_at_revision(path, rev)` backend command — available in both the Rust/Tauri and Node dev-server paths — fetches file bytes at any git revision so the viewer can compare `HEAD^` vs `HEAD`, across branches, across stashes, anything.

### 🌳 Folder tree view in the commit diff

The file list on the right of the commit diff gets a **flat ↔ tree** toggle. Tree mode collapses deeply-nested changes into an expandable folder tree with per-folder aggregates — number of files changed, total additions, total deletions — so a 200-file commit becomes navigable at a glance.

Click any folder to **filter the list** down to its descendants. A breadcrumb at the top surfaces the active filter with a clear (×) button.

The sidebar column is now **resizable** — drag its left edge, or double-click to reset. Width is clamped 180–600 px and persisted in `localStorage` so your layout sticks across sessions.

Keyboard navigation works throughout (arrow keys, Enter, Esc). Available in all 5 UI locales (English, French, Spanish, Portuguese, Chinese).

New `folder_diff` backend command (Rust/Tauri + Node/dev-server + TypeScript wrapper) builds the aggregated tree.

## Install

### Desktop app

Download the installer for your platform below (macOS, Linux, Windows).

Homebrew (macOS):

```bash
brew install --cask devlint/tap/gitwand
```

### npm packages

```bash
# CLI — auto-resolve conflicts from your terminal or CI
npm i -g @gitwand/cli

# MCP server — expose GitWand to Claude, Cursor, Windsurf, etc.
claude mcp add gitwand -- npx -y @gitwand/mcp

# Core engine — the underlying resolution library
npm i @gitwand/core
```

All three packages ship at `1.6.2`, published with npm provenance attestations.
